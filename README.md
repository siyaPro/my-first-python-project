# Student Pass Checker

This is a simple Python project created for learning purposes.

## Description
The program asks the user to enter a student's name and score, then checks whether the student has passed or failed the exam based on the score.

## How it works
- Takes input from the user (name and score)
- Uses if-else conditions
- Prints the result in the terminal

## How to run
Make sure Python is installed on your system, then run:

python main.py

## Purpose of this project
This project was created to practice:
- if / else conditions in Python
- taking input from the user
- using Git and GitHub to manage projects
